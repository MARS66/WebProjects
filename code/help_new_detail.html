冠通分销亮相厦门国际跨境电商展，助力卖家品牌出海

发布：东润商城 日期2021-07-19

由东润商城母公司纵腾集团受邀协办的首届国家级跨境电商交易会（跨交会），以“链接跨境全流域 共建电商新生态”为主题，于3月18日-20日在福州海峡国际会展中心隆重举行，旨在为传统外贸和跨境电商之间打造一座产销相通、共赢合作的桥梁，推动传统外贸数字化升级。 本届展会设置跨境电商综合平台展区、跨境电商服务商展区、跨境电商供货商展区和跨境电商品牌宣传展区四大展区，吸引全球33个跨境电商平台、35个全国出口型产业带、200+跨境电商服务企业、2000+外贸供货企业、6万多名跨境电商专业客商到场参会，3天展会意向成交额预计达35亿美元。 东润商城携旗下子公司冠通分销、云途物流、谷仓海外仓及跨呗金融纷纷亮相，向外界传递纵腾集团在商品分销、专线物流、海外仓储、供应链金融等方面强大的一体化服务能力。 福建省省长王宁莅临本届跨交会并前往纵腾集团展位，与纵腾集团总裁王钻和副总裁李聪一同深度交流本次展会的相关议题。
