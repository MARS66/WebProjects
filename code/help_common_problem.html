一、东润商城平台使用问题 1.1 东润商城海外仓货源一件代发是什么意思？ 这是一种新的跨境电商供应链服务，您无需备货，没有库存压力和滞销风险；您只需在我们东润商城平台上选品、下载商品数据包、上架到您的亚马逊/ebay/独立站等店铺，店铺出单后，到东润商城平台上单个/批量下单（或者授权店铺，SKU映射，关联载单系统），结款后将为您把包裹一件代发至您指定的客户手中。 1.2 东润商城海外仓货源一件代发的步骤是什么？ （1）注册东润商城会员 - 在东润商城分销官网按要求填写注册成为东润商城会员，开启分销之旅。 （2）下载产品数据包 - 根据您的选品需求，在东润商城分销官网免费下载您计划销售的对应商品数据包。 （3）刊登销售 - 将所下载的商品信息根据您的需求优化修改后，在亚马逊/eBay等店铺进行刊登销售。 （4）出单支付货款 - 店铺出单后，您在东润商城分销平台下单时付款，付款成功后系统将订单产品派送到您的终端客户地址。 （5）24小时出单号 - 东润商城分销系统接到您的下单请求后，在24小时内出物流跟踪号，妥投时效是2-8天。 （6）售后服务 - 依托海外仓优势，产生退货均退回东润商城海外仓，售后处理更加灵活。 总的售后原则： ① 如果遇到产品质量问题或者物流破损，提供相关图片或者视频证明后，东润商城将进行全额退款或者重新发货； ② 如果是客户无理由退货，例如出现地址错误、客户买了不想要的情况，那么客户出退回运费把货退回东润商城的仓库，东润商城将退给卖家45%-55%的货款。 1.3 怎么获得东润商城分销资格？ 访问东润商城的网站注册页面：https://www.gotenchina.com/register.html，你可以通过手机免费注册会员，只要填写认证完善个人信息，即可查看东润商城商城中产品的价格等具体信息，同时可下载商品数据包，获得东润商城分销的资格。 1.4 东润商城一件代发有什么优惠？ 东润商城根据会员每个月的销售额划分了5个不同等级，对应享受不同的订单折扣及会员权益，详见：https://www.gotenchina.com/help/44.html。 1.5 东润商城平台佣金怎么结算？ 东润商城分销平台是免费入驻的，采用出单结款、包邮一件代发的模式，您的下单价格即为您所需支付的所有费用。 1.6 交易过程中出了问题该找谁？ 如果您在使用东润商城平台的过程中遇到任何问题，可以直接联系您注册时系统为您分配的专属业务经理，谢谢。 二、产品问题 2.1 东润商城都有什么产品？ 东润商城平台上目前拥有家居和园艺、体育用品、健康和美容、乐器、汽车配件、玩具类、商业和工业、工艺品、计算机平板电脑和网络、手机配件、消费类电子产品、珠宝和手表、相机类这14条品类线，提供20000+ 东润商城平台上的产品有侵权风险吗？ 东润商城平台上的商品，除品牌商品外，其余的商品都是白牌产品，可以直接销售，不存在侵权风险(除去一款油桶和亚马逊禁卖的宝锋对讲机)。至于品牌商品，东润商城平台均已获得授权，如有计划分销的卖家可联系您的客户经理进行申请授权。 2.4 商品库存充足吗？ 东润商城平台上的商品都是经过大数据分析，备货到海外仓，大部分商品的库存都比较充足，但不排除某些商品比较热销，会导致出现暂时缺货的情况，一旦出现缺货情况，及时与您的客户经理沟通并进行库存管理。 2.5 如何申请品牌区产品分销权限？ 您可联系您的客户经理申请开通品牌区产品权限。 2.6 如何选择适合在亚马逊或ebay等平台上销售的商品？ 东润商城分销平台首页上有专门的“平台爆款”专区模块，点击“amazon ”或 “ebay” 图标即可进行访问。亦可点击如下链接进行访问选品： amazon 平台爆款商品：https://www.gotenchina.com/amazon.html ebay 平台爆款商品： https://www.gotenchina.com/ebay.html wish 平台爆款商品：https://www.gotenchina.com/wish.html 三、库存问题 出单了缺货怎么办，库存怎么管控？ 合作是需要有磨合期，我们针对在磨合期的新客户建议选库存多的产品进行推广，分销没办法100%保证完全不缺货，所以库存监控肯定是要做的，做的好的可以用东润商城平台载单，同时可以做跟踪号上传同步，库存的同步或者库存自行设置规则，后续稳定出单后可以合作备货。 另外，您可将感兴趣的产品加入收藏夹，然后在“个人中心”-“商品管理”，中，便可查看您所收藏的商品库存、预计可售天数、预计到货时间。 四、网站功能对接问题 3.1 东润商城是否支持ERP对接？ 东润商城暂不支持第三方ERP软件对接，但支持自主研发ERP技术API对接；市场上的各大ERP已在对接中，后续发布将另行通知，如您有相关需求，可反馈给您的客户经理。 3.2 东润商城是否支持一键刊登商品至店铺？ 东润商城暂无一键刊登功能。不过，东润商城平台上的商品数据包支持批量下载，您可以批量下载数据包，之后通过您使用的ERP软件进行一键刊登。 3.3 东润商城是否可以一键上传物流跟踪号至店铺？ 东润商城支持一键上传物流跟踪号至店铺，不过需要您使用东润商城的“平台载单”功能，该功能支持一键抓取Amazon / eBay / Wish平台店铺里的东润商城订单，以及自动上传跟踪号至店铺，详情可登录您的东润商城账户查看：《授权载单操作指引》。 五、物流问题 1、东润商城包裹运输的物流费用是怎么计算？ 东润商城平台上价格都是非偏远州包邮价格，如果偏远州看具体产品来评定物流，部分产品偏远州无法派送，建议屏蔽偏远州。 2、东润商城分销平台仓库地址在哪里？ 东润商城平台目前在美国有：美东仓-新泽西、美西仓-洛杉矶、美南仓-亚特兰大；英国有：伯明翰仓；德国仓：迪岑巴赫。更多海外仓信息可查看：https://www.gotenchina.com/help/33.html 3、东润商城发什么快递，会有跟踪号信息吗，时效怎样？ 东润商城美国仓主要是发Fedex，USPS、UPS为辅，英国主要走Hermes、Royal Mail（皇家物流），德国走DHL。东润商城平台上发货的包裹，基本都是24小时内出物流跟踪号，都可以在官网全程跟踪（一件多票需拆包的订单会稍微延后），非异常情况，美国是2-8天完成妥投，英国和德国是1-3天完成妥投。 4、东润商城平台退货的物流费用是怎么算的？ 若是因为商品破损等非无理由退货，您可在您的账户里发起退货申诉，系统将给您生成一个退货标签，您可将该退货标签提供给您的终端买家，让终端买家退回到东润商城海外仓即可，该退货费用，由东润商城分销平台承担。 若您的订单为无理由退货，则需要您联系网站客服申请退货标签，该退货费用将由系统自动测算，且由您自行承担。
