物流方式

美国仓支持美国本土包邮一件代发，东润商城系统将根据您的下单地址智能选择最佳的发货仓库。不过对于美国偏远区域，因距离海外仓较远、运输成本高，东润商城分销平台无法为您提供包邮服务，敬请谅解。   2、美国发货物流   Standard Shipping：参考时效5-7个工作日 Economy Shipping：参考时效3-10个工作日 Expedited Shipping：参考时效4-7个工作日   该美国平台标准物流产品内支持：Fedex、UPS、USPS等多种物流方案，东润商城系统将根据销售平台及商品尺寸、重量等因素自动匹配最佳发货物流。若下单街道地址中含有“PO Box”，则系统会根据您选择的发货物流进行审核判断。如果对应物流方式无法派送到含有“PO Box”的地址，则系统会将该订单判定为异常单，客服人员会通知修改地址或者取消订单。
